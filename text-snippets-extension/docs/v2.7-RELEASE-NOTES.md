# ✨ v2.7 - CHARACTER COUNTER

## 🎉 Nuova Feature: Conteggio Caratteri

Come richiesto dall'utente, ora ogni snippet mostra statistiche dettagliate!

---

## ✅ Cosa è stato aggiunto:

### 1. **Statistiche per ogni snippet nella lista**

Ogni card snippet ora mostra:
- 📏 **Caratteri**: Conteggio totale caratteri
- 📝 **Parole**: Numero di parole
- 📄 **Righe**: Numero di righe

```
┌────────────────────────────────────────┐
│ Ricetta Patate              [Ricette]  │
│ Ingredienti: 1kg patate...             │
│ 📏 1,245  📝 187  📄 23                 │
│ [📋 Incolla] [✏️ Modifica] [🗑️]         │
└────────────────────────────────────────┘
```

### 2. **Contatore live nel form di aggiunta**

Mentre digiti il testo, vedi in tempo reale:
```
Testo *
┌─────────────────────────────────────┐
│ Scrivi qui...                       │
│                                     │
└─────────────────────────────────────┘
         245 caratteri | 38 parole | 5 righe
```

### 3. **Contatore live nel modal di modifica**

Stesso contatore in tempo reale quando modifichi uno snippet esistente.

## 📊 Dettagli Tecnici:

### Calcolo Parole:
```javascript
const wordCount = text.trim()
  .split(/\s+/)
  .filter(w => w.length > 0)
  .length;
```
- Ignora spazi multipli
- Filtra stringhe vuote
- Conta parole reali

### Calcolo Righe:
```javascript
const lineCount = text.split('\n').length;
```
- Conta i newline
- Include riga vuota finale

### Formattazione:
```javascript
charCount.toLocaleString()
```
- 1234 → "1,234" (italiano)
- 1234567 → "1.234.567"
