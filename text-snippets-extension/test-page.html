🧪 Test Page - Estensione Snippet

📋 Come testare:

  1. Installa l'estensione v2.4
  2. Crea uno snippet di test (es. "TEST SNIPPET")
  3. Clicca in ogni campo qui sotto
  4. Usa Ctrl+Shift+V o tasto destro → Incolla snippet
  5. Verifica se il testo appare correttamente

Risultati attesi: Tutti i campi dovrebbero funzionare tranne "Non Supportato"

1. Textarea Standard ✅ DEVE FUNZIONARE

Gmail, form standard, commenti

2. Input Text ✅ DEVE FUNZIONARE

Campi nome, email, ricerca

3. ContentEditable Semplice ✅ DEVE FUNZIONARE

Editor WYSIWYG base

4. ContentEditable Complesso (stile Gemini) ✅ DEVE FUNZIONARE (v2.4)

Simula editor AI come Gemini, ChatGPT

5. ContentEditable + Event Handlers (stile React) ✅ DEVE FUNZIONARE (v2.4)

Editor con framework JavaScript

6. Textarea con Event Validation ✅ DEVE FUNZIONARE

Form con validazione JavaScript

7. Input Disabled ❌ NON PUÒ FUNZIONARE

Campo disabilitato (corretto che non funzioni)

8. Textarea Readonly ❌ NON PUÒ FUNZIONARE

Campo readonly (corretto che non funzioni)

9. ContentEditable False ❌ NON PUÒ FUNZIONARE

Div non editabile

Questo div non è editabile

🐛 Se qualcosa non funziona:

  1. Apri DevTools (F12)
  2. Vai alla tab Console
  3. Riprova a incollare
  4. Cerca messaggi che iniziano con [Snippet]
  5. Copia TUTTI i log e mandali allo sviluppatore

Logs da cercare:

  • [Snippet] Starting paste operation
  • [Snippet] Target element: {...}
  • [Snippet] Trying Clipboard API
  • [Snippet] Text copied to clipboard
  • [Snippet] Paste event dispatched
